Nav Item Component

Demonstration of the nav-item component with all states

Component States

The nav-item component has 4 states: default, hover, active, and active+hover. Hover effects extend 4px around the item.

Component Specifications

  • Height: 24px
  • Max Width: 240px (fills container)
  • Icon: 16px Ã— 16px in 24px square container
  • Label: 8px spacing from icon, label/medium typography
  • Default: Text color neutral-900
  • Active: Text color brand-600, label/medium-emphasized
  • Hover: Neutral-50 background (#EBEEF2), extends 4px around item
  • Active + Hover: Maintains active styling with hover background
