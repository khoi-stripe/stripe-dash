Usage Guidelines

Display Styles

  • Use for hero sections and major page titles
  • Display XLarge for main landing page heroes
  • Display Large for section heroes
  • Consider responsive scaling on mobile

Heading Hierarchy

  • Heading XLarge: Main page titles (H1)
  • Heading Large: Major section titles (H2)
  • Heading Medium: Subsection titles (H3)
  • Heading Small: Minor headings (H4-H6)

Body Text

  • Body Large: Lead paragraphs and important content
  • Body Medium: Standard paragraph text
  • Body Small: Captions and secondary content
  • Use emphasized variants for important information

Labels & Code

  • Labels: Form labels, UI component text
  • Code: Technical content, inline code
  • Match label size to corresponding input size
  • Use monospace for all technical content
