How to Use

Dotted Versions:

  • .figma-dotted-background - Standard 24px grid
  • .figma-dotted-background-alt - 20px grid alternative
  • .figma-dotted-background-subtle - 16px grid, most subtle
  • .figma-page-background - Full-page background with fixed attachment

Gradient Versions:

  • .figma-gradient-background - Gradient + standard dots
  • .figma-gradient-clean - Pure gradient, no dots
  • .figma-gradient-subtle - Gradient + subtle dots
